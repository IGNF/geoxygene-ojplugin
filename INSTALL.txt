
-----------------  How to install   -----------------

Set up OpenJump as a prerequisite.

1. Download the plugin

2. Copy the whole folder contents into your 
    lib/ext directory of your OpenJump installation.

3. Start openjump. 

4. You will now see a new item menu in "Extensions" called "GeOxygene" 

If any problem encountered, you can report it
	on the GeOxygene users list :
	oxygene-project-users (at) lists.sourceforge.net
